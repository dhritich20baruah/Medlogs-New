# Medlogs
## Description
This repository contains the code for the vversion 2 of the Medlogs android application which is published in the google play store. The app is used by patients and care givers to track medication. Record and store blood sugar and hypertension readings. Also store information about health care providers.
## Installation
To install the application in your local machine. First clone the repo. Then run 
npm install
This application is built in expo, so you will need to install expo go in your mobile device. Both the mobile device and the computer should be on the same network.
Then run npm run start. 
You will be provided with a qr code which you can scan with the expo go app to use your mobile device as an emulator.
